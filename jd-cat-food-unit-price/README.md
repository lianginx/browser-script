# 京东商品单价计算

在京东购物车页面自动识别猫粮、大米、小米等粮食类商品，并根据标题中的重量信息计算单价，便于横向比较不同商品的价格。

## 使用方式

- [点击安装](https://raw.githubusercontent.com/lianginx/browser-script/refs/heads/master/jd-cat-food-unit-price/jd-cat-food-uni-price.user.js)

1. 在 Tampermonkey、Violentmonkey 等用户脚本管理器中安装脚本
2. 打开京东购物车页面
3. 等待购物车内容加载完成后查看每个猫粮商品旁的单价结果

## 开发测试

在脚本目录运行：

```bash
node test.js
```

测试脚本直接复用 `jd-cat-food-uni-price.user.js` 导出的纯函数，校验重量提取、单价计算和商品匹配的典型案例：

- `10斤*4`、`5斤*4`、`2.5kg*4` 这类乘数写法
- `10kg/20斤`、`40斤整箱装` 这类汇总重量写法
- 当前已整理到 `test.js` 的案例

## 功能

- 扫描购物车中的商品标题，当前处理猫粮、大米、香米、小米等粮食类商品
- 自动提取标题中的重量信息，支持 `kg`、`g`、`斤`
- 计算并展示每公斤和每 `500g` 的价格
- 将单价直接插入到商品价格区域附近，比较时无需手动换算

## 适用页面

- `https://cart.jd.com/cart_index*`

## 注意事项

- 依赖商品标题中存在明确重量信息
- 当前主要处理标题包含“猫粮”“大米”“香米”，以及“黄小米”“山西小米”等明确粮食类小米商品
- 不会将“小米手机”等品牌名称误判为粮食类商品
- 如果京东购物车页面 DOM 结构发生变化，脚本可能需要同步调整选择器
